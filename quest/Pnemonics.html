🧠 Mnemonics for Ionic Equilibrium

Immersive learning with dedicated, realistic visuals.

Strong Acids

Hi Hi Strong Super Clean Night

HI, HBr, HCl, H₂SO₄, HClO₄, HNO₃

Memory Hack

Link ‘Hi’ with Halogen acids (HCl, HBr, HI). ‘Strong Super Clean Night’ = Nitric, Sulfuric, Perchloric → easy night revision.

Strong Bases

Never Keep Strong Bases Close

NaOH, KOH, Sr(OH)₂, Ba(OH)₂, Ca(OH)₂

Memory Hack

Think of ‘Never Keep’ as sodium & potassium (Na, K). ‘Strong Bases Close’ = calcium, barium, strontium → grouped together in periodic table.

Weak Acids

Apple Candy Lingers Forever

Acetic, Carbonic, Lactic, Formic

Memory Hack

Visualize eating sour apple candy → sourness = weak acids.

Weak Bases

Never Amuse Weak Chemistry

NH₃ (Ammonia), Amines

Memory Hack

Ammonia = Never Amuse. Carbonates bubble weakly in acids → Weak Chemistry.

Solubility Product

All Pretty Beautiful Crystals

Example: AgCl < PbCl₂ < BaSO₄

Memory Hack

Imagine a jewelry box: All Pretty Beautiful Crystals = Silver, Lead, Barium, Calcium.

Buffer System

Apple sauce / Banana shake

Acidic / Basic buffers

Memory Hack

Apple sauce = acidic, Banana shake = basic → easy food analogy.

Henderson–Hasselbalch

Please Keep All Logs Handy

pH = pKa + log([A⁻]/[HA])

Memory Hack

LOG equation → keep logbook handy.

Common Ion Effect

Adding Common Ions Suppresses

Reduces dissociation.

Memory Hack

Adding common people to a room suppresses conversations → less dissociation.

Degree of Dissociation (α)

Alpha Always Defines Ionization

α = (dissociated) / (initial)

Memory Hack

Greek Alpha = beginning → defines how much ionization begins.

Ostwald’s Dilution Law

Only Dilution Loves Weak Acids

Dissociation increases with dilution.

Memory Hack

Weak acids become stronger with more dilution → easier dissociation.
